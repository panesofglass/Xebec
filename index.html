Xebec

Imagine a world where you could have your F# fun without the msbuild pain, misery, and suffering ...

View on GitHub Download .zip Download .tar.gz

Note: All documentation is preliminary and will be incrementally improved.

Xebec aims to fulfil a dream, a dream where you don't have to use and fight against msbuild. It also allows you to use other xebec packages as references which will be build if necessary as part of the build process.

It takes inspiration from package management schemes from language like Rust and Elixir.

A very simple example would be as following toml file:

[project]
name = "helloworld"
version = "0.1.0"
authors = ["Your Name <you@example.com>"]
source = ["main.fs"]

Xebec allows you to use a github repo as a project dependency rather than a just nuget package, or dll:

[dependencies]
tlh = { git = "https://github.com/7sharp9/TheLurkingHorror" }

The github repo will be downloaded and compiled as part of your build.

The following types of dependencies are also supported git, nuget, local, and dll can be specified

Command Line

Xebec can also be used from the command line, the following commands can be used:

new

Creates a new xebec project

build

The build option allows you to compile your current project

run

The new option allows you to compile and run your xebec project

clean

The clean option clean up and temporary files and target directory/s

test

Run the tests

doc

Compiles project and produces documentation

update

Update dependencies
